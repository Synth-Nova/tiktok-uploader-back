   • IP Address + Geolocation — должны совпадать со страной аккаунта
   • Timezone — должен соответствовать IP
   • Language — должен соответствовать стране
   • Webdriver = true — 100% детект автоматизации
   • Canvas/WebGL Hash — уникальность устройства

🟡 ВАЖНО (анализируют паттерны):
   • User-Agent — должен быть реалистичным и актуальным
   • Screen Resolution — только популярные размеры
   • Fonts — должны соответствовать ОС и региону
   • Hardware — реалистичные комбинации CPU/RAM/GPU

🟢 ДОПОЛНИТЕЛЬНО (для полноты профиля):
   • Audio fingerprint
   • Plugins
   • Connection type
   • WebRTC IPs

🎉 Видеофайлы ЯВЛЯЮТСЯ УНИКАЛЬНЫМИ для антифрода TikTok!

Причины:
  1. ✅ Разный язык озвучки = 100% разный audio fingerprint
  2. ✅ Разные субтитры = изменённый visual fingerprint
  3. ✅ Разные размеры и битрейты = разные технические хэши
  4. ✅ Разная длительность = дополнительное отличие

⚠️  ПОТЕНЦИАЛЬНЫЙ РИСК:
    Если TikTok использует продвинутый CV (Computer Vision) для 
    сравнения видеоряда БЕЗ субтитров и звука, теоретически может 
    определить похожесть. Однако на практике такой глубокий анализ 
    применяется редко.

💡 РЕКОМЕНДАЦИЯ ДЛЯ 100% ЗАЩИТЫ:
    При клонировании для разных аккаунтов ОДНОЙ страны добавлять:
    • Сдвиг яркости/контраста: ±3-5%
    • Обрезка начала/конца: 0.1-0.5 сек
    • Лёгкий шум: 1-2%
    • Изменение скорости: 98-102%
